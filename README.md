<p align="center">
  <code>glasshouse</code> is a minimal, single glass card landing page <a href="https://gohugo.io">Hugo</a> theme.
</p>

---

### Usage

```
git submodule add git@github.com:huhndev/glasshouse.git themes/glasshouse
```

Set `theme = "glasshouse"` in your `hugo.toml` and configure the card through `[params]`. Demos: [as215236.huhn.dev](https://as215236.huhn.dev), [henhouse.im](https://www.henhouse.im).
